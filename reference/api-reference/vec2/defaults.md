# Defaults

### `zero`

Macro: Shorthand for writing `new vec2(0, 0)`

```gml
pos = zero;
//pos.x = 0
//pos.y = 0
```

### `one`

Macro: Shorthand for writing `new vec2(1, 1)`

```gml
pos = one;
//pos.x = 1
//pos.y = 1
```



### `up`

Macro: Shorthand for writing `new vec2(0, 1)`

```gml
pos = up;
//pos.x = 0
//pos.y = 1
```

### `down`

Macro: Shorthand for writing `new vec2(0, -1)`

```gml
pos = down;
//pos.x = 0
//pos.y = -1
```

### `left`

Macro: Shorthand for writing `new vec2(-1, 0)`

```gml
pos = left;
//pos.x = -1
//pos.y = 0
```

### `right`

Macro: Shorthand for writing `new vec2(1, 0)`

```gml
pos = right;
//pos.x = 1
//pos.y = 0
```
